# Семейный бюджет

Небольшой учебный веб-проект с приветственным лендингом и калькулятором семейного бюджета.

## Что умеет

- приветственный лендинг с описанием проекта;
- калькулятор доходов и расходов (включая судебные расходы);
- расчет итогового баланса;
- подсветка отрицательного баланса;
- переключение светлой/темной темы (с сохранением выбора в `localStorage`).

## Структура проекта

```text
.
├── index.html
├── styles/
│   └── main.css
├── scripts/
│   └── main.js
└── README.md
```

## Запуск

1. Откройте файл `index.html` в браузере.
2. Введите значения в поля доходов и расходов.
3. Нажмите кнопку **Рассчитать бюджет**.
